## Gantry v0.1.0 (Preview)

Initial public preview release of Gantry core.

### Highlights
- Multi-platform bridge path for Cursor, Windsurf, and VS Code automation.
- Telegram-first command surface with additional Discord, Email, Feishu, and HTTP adapters.
- Security guardrails for inbound exfiltration patterns and outbound secret redaction.
- Release quality gate with lint, typecheck, build, smoke checks, and secret scan.

### Known Limitations
- `v0.x` preview: interfaces and behavior may change before `v1.0.0`.
- IDE selector reliability can vary by product/version and may require updates.
- Some adapter features remain best-effort and are documented as such in README tables.
